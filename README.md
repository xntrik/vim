# vim

My mostly terrible vim dot files. 

To setup:

```console
$ cd ~/
$ git clone --recursive https://github.com/xntrik/vim .vim
$ cd .vim
$ make update
$ make install
```

To add a new vim plugin:

```console
$ cd bundle
$ git submodule add <git>
```
